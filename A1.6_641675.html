Justificacin para eliminar los outliers

En este anlisis, hemos identificado y decidido eliminar dos tipos de outliers en la columna "Tiempo" debido a su naturaleza anmala. Estos outliers representan casos que no son relevantes para el anlisis, ya que distorsionan los resultados y afectan la precisin del modelo. A continuacin, se explica la razn de la eliminacin de cada uno:

  1. Tiempo negativo (< 0 horas):

    • Un valor negativo en "Tiempo" indica que la entrega de la tarea fue registrada despus del cierre de la bandeja, lo cual es un error, al menos que tuvieramos ms informacin sobre si hay tiempo extra despus del cierre. Este valor no tiene sentido y debe ser eliminado del conjunto de datos.

  2. Tiempo mayor a 300 horas (> 300 horas):

    • Un valor mayor a 300 horas indica que la tarea fue entregada con una antelacin excesiva, especficamente con ms de 12 das antes de la fecha de cierre. Este tipo de entrega es inusual y podra ser el resultado de un error en la recoleccin de datos. El valor excede el rango razonable de entrega y debe eliminarse para evitar que distorsione el anlisis.

Dicho modelo no asume ningn tipo de relacin entre la variable de entrada y la de salida. Usaremos la funcin KNeighbotsRegressor de sklearn.neighbors para generar el modelo, y la funcin fit para entrenarlo. Probaremos en este caso con 15 vecinos.

In[81]:
# Importar libreras
from sklearn.neighbors import KNeighborsRegressor as KNN


# Definir los valores de k a probar
k_values = [1, 5, 10, 15, 20]
models = {}

# Entrenar modelos con diferentes valores de k
for k in k_values:
    # Entrenar el modelo KNN con el valor de k
    models[k] = KNN(n_neighbors=k).fit(Xtemp_train, Y_train)
    
    # Estimar el valor de Y en los datos de prueba
    yHat_mk = models[k].predict(Xtemp_test)
    
    # Calcular RSS (Residual Sum of Squares)
    RSS_mk = sum((list(Y_test) - yHat_mk)**2)
    
    # Calcular RSE (Raz del Error Cuadrtico Medio)
    RSE_mk = np.sqrt(RSS_mk / (len(Y_test)-1))  # Manteniendo el clculo como lo proporcionaste
    
    # Imprimir el valor de RSE para la iteracin actual
    print(f"RSE para k={k}: {RSE_mk}")
